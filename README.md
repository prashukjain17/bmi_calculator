## Project Overview

This project is a console-based BMI Calculator implemented in Python. It provides:
- Robust input validation,
- BMI computation and categorization,
- Personalized suggestions and motivational messages,
- A simple menu so the user can calculate BMI, read about BMI, or exit.

It's ideal for practical assignments, quick demos, or as a basis for GUI/web extensions.

---

## Features

- Interactive CLI with friendly prompts.  
- Accepts both numeric menu choices (`1`, `2`, `3`) and text commands like `calculate`, `about`, `exit`.  
- Input validation for numeric/positive height and weight.  
- Clean output formatting with BMI to two decimal places.  
- Category-specific suggestions + random motivational quote.

## Technical Stack

- Language: **Python 3.8+**
